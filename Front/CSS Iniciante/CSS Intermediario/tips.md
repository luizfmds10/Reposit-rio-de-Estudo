```
a:hover {
background: purple
}
```
(Hover é quando passa o mouse por cima de algum elemento, no exemplo A (link) é um elemento que mudará quando passar o mouse por cima de acordo com o que você coloca dentro)

```
a:active {
background: purple
}
```

(Active é na hora do click, as mudanças feitas dentro do style são aplicadas quando o CLICK é efetuado)
